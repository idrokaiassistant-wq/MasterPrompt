# Foydalanuvchi uchun qo'llanma (User Guide)

**Dastur nomi:** Prompt Academy AI v1.0
**Muallif:** Nurmurodov Damirjon Salohiddin o'g'li

---

## 1. Boshlang'ich qo'llanma

### Tizimga kirish
Platforma barcha qurilmalarda (kompyuter, planshet, telefon) ishlaydi. Brauzeringiz orqali manzilga kiring.

### Ro'yxatdan o'tish
Hozirgi versiyada (v1.0) ro'yxatdan o'tish talab qilinmaydi. Siz to'g'ridan-to'g'ri tizim imkoniyatlaridan foydalanishni boshlashingiz mumkin.

### API Kalitlarini sozlash
Dasturdan to'liq foydalanish uchun (AI modellar bilan ishlash):
1. **Sozlamalar (Settings)** bo'limiga o'ting.
2. O'zingizga kerakli AI modeli uchun kalitni kiriting:
   - **Google Gemini API Key:** Bepul va tezkor modellar uchun (Tavsiya etiladi).
   - **OpenRouter API Key:** GPT-4o, Claude va boshqa modellar uchun.
3. Kalitlar xavfsiz shifrlangan holda saqlanadi.

---

## 2. Asosiy funksiyalar

### 1. Prompt Yaratish (Generate)
Bu bo'limda siz o'z kasbingizga mos professional promptlarni yaratishingiz mumkin.
- **Kirish:** Maqsadli rol va vazifani yozing (masalan: "Dasturchi, Python funksiya yozish").
- **Natija:** Tizim sizga mukammal tuzilgan promptni taqdim etadi.

### 2. O'qituvchi (Teacher Agent)
Interaktiv ta'lim rejimi.
- **Suhbat:** O'qituvchi bilan jonli suhbatlashing.
- **Tahlil:** Yozgan promptingizni yuboring, o'qituvchi uni 10 ballik tizimda baholaydi va xatolarini ko'rsatadi.
- **Ovozli kirish:** Savollarni ovoz orqali berish imkoniyati mavjud.

### 3. Shablonlar (Templates)
Tayyor qoliplardan foydalaning.
- **G'oyani Baholash:** Startap g'oyalarini 10 ballik tizimda tahlil qilish.
- **Maqola yozish, Kod yozish** va boshqa ko'plab tayyor shablonlar.

## 3. Tez-tez beriladigan savollar (FAQ)

**Savol: Dastur bepulmi?**
Javob: Ha, dasturning o'zi bepul. Faqat AI modellaridan (Gemini, GPT) foydalanish uchun ularning o'z tariflari amal qiladi (Gemini Flash ko'pincha bepul).

**Savol: Ma'lumotlarim xavfsizmi?**
Javob: Ha, API kalitlaringiz faqat sizning brauzeringiz va server o'rtasida shifrlangan holda uzatiladi.

**Savol: Qaysi tillarni qo'llab-quvvatlaydi?**
Javob: O'zbek, Ingliz, Rus va Turk tillari to'liq qo'llab-quvvatlanadi.
